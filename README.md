## Domestic Animation Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/3c75ba17a36d49fbbb9924ca18fcade1/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Chinese Animation Pioneers
5. Forum
6. Best Chinese Animation
7. Historical Evolution
8. Announcement Information
9. Knowledge Window
10. Personal Center
11. Chinese Animation Pioneers Collection
12. Chinese Animation Pioneers Messages
13. Best Chinese Animation Collection
14. Best Chinese Animation Messages


**Administrator Functions:**
1. Login
2. Personal Center (change password, personal information)
3. Administrator Management
4. Basic Data Management (Chinese Animation Pioneers type management, Best Chinese Animation type management, content type management, announcement type management, knowledge type management, animation type management)
5. Chinese Animation Pioneers Management
6. Chinese Animation Pioneers Message Management
7. Chinese Animation Pioneers Collection Management
8. Forum Management
9. Best Chinese Animation Management
10. Best Chinese Animation Message Management
11. Best Chinese Animation Collection Management
12. Historical Evolution Management
13. Announcement Information Management
14. User Management
15. Knowledge Window Management
16. Carousel Image Management
